# Ooniverse
A simluation of a 2-dimensional universe from from Big Bang to (hopefully) the emergence of life and beyond.

## Build

0. `mkdir build`
1. `cd build` from the repository root
2. `cmake ..`
3. `make`

Compiled executable binary will be at `/build/Ooniverse`.

## Information by path

### .vscode

- `c_cpp_properties.json`
  - This file is for configuring intellisense

- `settings.json`
  - This file is for configuring the compiler, linker, and so-on

- `launch.json`
  - This file configures how the compiled program is executed
